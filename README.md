# build-with-ai

개인 개발 작업 방식, 도구 설계, 의사결정 흐름을 기록.

AI와 같이 일하다 보면 자연스럽게 쌓이는 규칙·도구·회고가 있다. 프로젝트 repo 안에 두면 다음 작업에서 다시 꺼내 쓰기 어렵기 때문에, 프로젝트와 독립적으로 한 곳에 모아두는 공간을 따로 만들었다.

## 구조

```
build-with-ai/
├── README.md
└── workflow/      # 개발 워크플로우, 도구 설계, 의사결정 흐름
```

향후 확장: `tools/`, `decisions/`, `experiments/` 등.
